## Reporting Match FTL - v1.0

Première version stable de l'application pour générer des résumés de replays StarCraft 2.

- Interface guidée (sélection des replays, nom des clans, assignation joueurs + alias)
- Résumé clair avec score global, matches 2v2/1v1 séparés
- Enregistrement du résumé non fonctionnel pour le moment

Il suffit de glisser les fichiers .SC2Replay dans un dossier et lancer l'outil !
